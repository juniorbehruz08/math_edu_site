Logarithmic Functions — Complete Introduction

A logarithmic function is a function that contains a logarithm. It is closely connected to exponential functions. Exponential functions show growth using powers, while logarithmic functions help us find the power behind a value.

\(f(x)=\log_b x\)

In this function, \(b\) is the base and \(x\) is the input. The function asks: “What power of \(b\) gives \(x\)?”

Main idea: A logarithmic function turns an input into the exponent needed to create that input.

\(a\) stretches or reflects the graph, \(b\) changes the base, \(h\) moves the graph left or right, and \(k\) moves it up or down.

2Basic Form

The simplest logarithmic function is called the parent logarithmic function.

\(f(x)=\log_b x\)

This means the function gives the exponent needed to get \(x\) from the base \(b\).

Base

The base is \(b\). It must be positive.

Input

The input is \(x\). It must be greater than zero.

Output

The output is \(f(x)\), which is an exponent.

Graph

The graph is a curve with a vertical asymptote.

3Connection with Exponential Functions

Logarithmic functions and exponential functions are inverse functions. This means they undo each other.

\(y=\log_b x\)

This is the same relationship as:

\(b^y=x\)
Simple meaning: If \(y=\log_b x\), then \(y\) is the exponent that makes \(b^y=x\).
Logarithmic form Exponential form Meaning
\(y=\log_2 x\) \(2^y=x\) \(y\) is the exponent of \(2\).
\(y=\log_3 x\) \(3^y=x\) \(y\) is the exponent of \(3\).
\(y=\log_{10}x\) \(10^y=x\) \(y\) is the exponent of \(10\).

4Domain and Range

The domain tells which input values are allowed. The range tells which output values are possible.

\(f(x)=\log_b x\)
Domain
\(x>0\)

The input must be positive because logarithms of zero or negative numbers are not defined in basic real-number algebra.

Range
\((-\infty,\infty)\)

The output can be any real number.

Remember: The graph of the parent logarithmic function exists only for positive \(x\)-values.

5Vertical Asymptote

A vertical asymptote is a vertical line that the graph gets closer and closer to, but does not touch. For the parent logarithmic function, the vertical asymptote is:

\(x=0\)

This happens because the logarithm is not defined at \(x=0\). The graph can get very close to the \(y\)-axis, but it cannot cross it.

Important: For \(f(x)=\log_b x\), the \(y\)-axis is the vertical asymptote.

7Increasing and Decreasing Graphs

The base decides whether the logarithmic function increases or decreases.

If \(b>1\)

The logarithmic function increases from left to right.

\(f(x)=\log_2x\)
If \(0<b<1\)

The logarithmic function decreases from left to right.

\(f(x)=\log_{\frac{1}{2}}x\)
Simple rule: Bases greater than \(1\) give increasing graphs. Bases between \(0\) and \(1\) give decreasing graphs.

8Transformations

A logarithmic graph can move, stretch, shrink, or reflect. A common transformed logarithmic function is:

\(y=a\log_b(x-h)+k\)
Part Effect
\(a\) Stretches, shrinks, or reflects the graph.
\(b\) Changes the base and affects the curve shape.
\(h\) Moves the graph left or right. The vertical asymptote becomes \(x=h\).
\(k\) Moves the graph up or down.
Important: In \(y=a\log_b(x-h)+k\), the vertical asymptote is \(x=h\), not always \(x=0\).

9Real-Life Meaning

Logarithmic functions are useful when values grow very quickly, but we want to describe them on a smaller scale. They help compress large numbers into easier numbers.

Sound

Decibels use logarithmic thinking to measure sound intensity.

Earthquakes

Earthquake scales use logarithmic ideas to compare very large amounts of energy.

Computer science

Logarithms appear in algorithms, data structures, and binary search.

Science

Logarithmic scales are used when values vary across very large ranges.

10Common Mistakes

1. Forgetting the domain

A logarithmic function cannot take zero or negative inputs in basic real-number algebra.

2. Thinking the graph crosses the \(y\)-axis

The parent logarithmic graph gets close to the \(y\)-axis, but it does not cross it.

3. Confusing logarithmic and exponential graphs

Exponential graphs usually have a horizontal asymptote. Logarithmic graphs have a vertical asymptote.

4. Ignoring the base

The base controls whether the graph increases or decreases.

11Final Summary

  1. A logarithmic function has the basic form \(f(x)=\log_bx\).
  2. The base must be positive and cannot be \(1\).
  3. The input \(x\) must be positive.
  4. The domain of \(f(x)=\log_bx\) is \(x>0\).
  5. The range is all real numbers.
  6. The parent graph has a vertical asymptote at \(x=0\).
  7. The graph crosses the \(x\)-axis at \((1,0)\).
  8. If \(b>1\), the function increases.
  9. If \(0<b<1\), the function decreases.
  10. Logarithmic functions are inverses of exponential functions.
