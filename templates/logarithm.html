Logarithms — Introduction

A logarithm is a mathematical idea that helps us find an exponent. In simple words, a logarithm answers this question:

\(\text{To what power should we raise the base?}\)

Logarithms are closely connected to exponents. If you understand powers such as \(2^3=8\), logarithms become much easier to understand.

Main idea: A logarithm is the reverse way of thinking about an exponent.

1Remembering Exponents

Before learning logarithms, we need to remember what an exponent means. An exponent tells how many times a base is multiplied by itself.

\(2^3=8\)

This means:

\(2\times2\times2=8\)
Base

In \(2^3\), the base is \(2\).

Exponent

In \(2^3\), the exponent is \(3\).

Result

The result of \(2^3\) is \(8\).

Meaning

The base \(2\) is multiplied by itself \(3\) times.

2What Does a Logarithm Mean?

A logarithm asks for the exponent. For example:

\(\log_2 8=3\)

This means:

\(\text{2 raised to what power gives 8?}\)

The answer is \(3\), because:

\(2^3=8\)
Simple explanation: \(\log_2 8=3\) and \(2^3=8\) say the same relationship in two different ways.

3Logarithmic Form and Exponential Form

The most important skill in logarithms is changing between logarithmic form and exponential form.

\(\log_b a=c\)

This means:

\(b^c=a\)
Symbol Name Meaning
\(b\) Base The number that is raised to a power.
\(a\) Argument The number inside the logarithm.
\(c\) Exponent The power we are trying to find.
Important: If a logarithm looks confusing, rewrite it as an exponent first.

4How to Read Logarithms

The notation \(\log_2 8\) is read as “log base 2 of 8.”

\(\log_2 8=3\)

You can read it like this:

The base is \(2\).
The number inside the logarithm is \(8\).
The logarithm asks which exponent gives \(8\).
Since \(2^3=8\), the logarithm equals \(3\).

5Basic Examples

These examples show the basic meaning of logarithms. For now, focus only on understanding the meaning, not on complicated solving methods.

Logarithm Equivalent Exponential Form Reason
\(\log_2 8=3\) \(2^3=8\) Because \(2\) raised to \(3\) gives \(8\).
\(\log_3 27=3\) \(3^3=27\) Because \(3\) raised to \(3\) gives \(27\).
\(\log_5 25=2\) \(5^2=25\) Because \(5\) raised to \(2\) gives \(25\).
\(\log_{10}1000=3\) \(10^3=1000\) Because \(10\) raised to \(3\) gives \(1000\).

6Parts of a Logarithm

A logarithm has several parts. Understanding these parts makes the notation much less scary.

\(\log_b a=c\)
Base

The base is \(b\). It tells which number is being raised to a power.

Argument

The argument is \(a\). It is the number inside the logarithm.

Value

The value is \(c\). It is the exponent we get as the answer.

Connection

The connection is \(b^c=a\).

7Common Logarithms

Some logarithms are used so often that they have special names.

Type Notation Meaning
Common logarithm \(\log x\) Usually means base \(10\): \(\log_{10}x\).
Natural logarithm \(\ln x\) Means base \(e\): \(\log_e x\), where \(e\approx2.718\).
Binary logarithm \(\log_2 x\) Base \(2\), often used in computer science.

In beginner algebra, when the base is not written in \(\log x\), it usually means base \(10\).

8When Is a Logarithm Valid?

Logarithms have conditions. These conditions tell us when a logarithm is allowed in real-number algebra.

The base must be positive
\(b>0\)
The base cannot be one
\(b\ne1\)
The argument must be positive
\(a>0\)
The result can be any real number

A logarithm can have a positive, negative, or zero value.

Important: In this lesson, we only work with logarithms in the real number system.

9The Main Idea to Remember

Logarithms and exponents are two ways to describe the same relationship. One form focuses on the exponent, and the other form focuses on the result.

\(2^3=8\)

This can also be written as:

\(\log_2 8=3\)
Memory trick: The answer to a logarithm is an exponent.

10Final Summary

  1. A logarithm asks for an exponent.
  2. \(\log_b a=c\) means \(b^c=a\).
  3. The base is the number being raised to a power.
  4. The argument is the number inside the logarithm.
  5. The value of the logarithm is the exponent.
  6. The base must be positive and cannot be \(1\).
  7. The argument must be positive.
  8. Logarithms are the reverse way of thinking about exponents.
