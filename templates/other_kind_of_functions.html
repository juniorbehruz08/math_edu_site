0Before You Start: The Language of Functions

A function is a precise mathematical rule that assigns exactly one output to every valid input. Think of it as a machine: you feed it a number, it follows a fixed process, and it produces one result. We usually write:

\(y=f(x)\)

This notation means “the output \(y\) depends entirely on the input \(x\).” The graph of a function is a visual map of every possible input-output pair plotted on the coordinate plane. If a vertical line ever crosses a graph more than once, it fails the vertical line test and is not a function.

Input

The value you put into the function. Usually this is \(x\), also called the independent variable because you choose it freely within the domain.

Output

The value the function gives back. Usually this is \(y\) or \(f(x)\), also called the dependent variable because its value depends on \(x\).

Graph

A visual picture of all points \((x,y)\) that satisfy the function rule. The shape reveals behavior: increasing/decreasing intervals, symmetry, bounds, and asymptotic trends.

1Domain, Range, Intercepts, and Intersections

Domain

All allowed \(x\)-values. Ask: “Which inputs are legal?” Restrictions come from division by zero, even roots of negative numbers, or logarithms of non-positive values.

Example: In \(\frac{1}{x-2}\), \(x=2\) makes the denominator zero, so it is excluded from the domain.

Range

All possible \(y\)-values the function can actually produce. Ask: “Which outputs can happen?” The range is often found by analyzing the graph’s highest/lowest points or asymptotic limits.

Example: For \(x^2\), squaring any real number never gives a negative result, so the range is \(y\ge0\).

\(x\)-intercept

Where the graph crosses the \(x\)-axis. At these points, the output is zero, so set \(y=0\) or \(f(x)=0\) and solve for \(x\). These are also called roots or zeros.

\(y\)-intercept

Where the graph crosses the \(y\)-axis. This happens when the input is zero, so set \(x=0\) and evaluate \(f(0)\). A function can have at most one \(y\)-intercept.

When two functions intersect, they share the exact same coordinates at that point. This means \(f(x)=g(x)\) at the intersection. Solve this equation for \(x\), then substitute back into either function to find the matching \(y\)-value.

Quadratic Functions: Complete Case-by-Case Lesson

Change \(a\), \(b\), and \(c\) to see how the parabola \(y=ax^2+bx+c\) moves. Watch especially how \(a\) changes direction and width, and how \(c\) moves the \(y\)-intercept.

\(y = ax^2 + bx + c\)
1What is a quadratic function?

A quadratic function is a polynomial of degree \(2\). The word “quadratic” comes from the Latin “quadratus,” meaning square, because the variable is squared. The highest power of \(x\) is exactly \(2\), which guarantees the graph is always a symmetric U-shaped curve called a parabola. The condition \(a \neq 0\) is essential; if \(a=0\), the \(x^2\) term vanishes and the function becomes linear.

\(f(x)=ax^2+bx+c,\qquad a \neq 0\)
  • \(a\) controls whether the parabola opens upward or downward. Positive \(a\) creates a “smile” shape (minimum), negative \(a\) creates a “frown” shape (maximum).
  • \(a\) also controls how wide or narrow the parabola is. Larger \(|a|\) compresses the graph vertically, making it steeper and narrower. Smaller \(|a|\) stretches it, making it wider and flatter.
  • \(b\) helps determine the position of the vertex and axis of symmetry. Together with \(a\), it shifts the parabola left or right according to \(x=\frac{-b}{2a}\).
  • \(c\) is the \(y\)-intercept because \(f(0)=c\). It vertically shifts the entire parabola without changing its shape or direction.
A quadratic is not just “a curve.” It is a highly structured graph with predictable features: a single vertex (turning point), a vertical axis of symmetry, a consistent opening direction, up to two real roots, and a bounded range on one side. These properties make quadratics foundational in physics, economics, and optimization.
2The three main forms of a quadratic

Each form highlights different features of the same parabola. Converting between them is a core algebraic skill that reveals hidden information about the graph.

Case Form A: Standard Form

\(f(x)=ax^2+bx+c\)
  • Best for identifying coefficients \(a,b,c\) directly.
  • \(c\) gives the \(y\)-intercept instantly.
  • Vertex uses \(x=\frac{-b}{2a}\), then substitute to find \(y\).

Case Form B: Vertex Form

\(f(x)=a(x-h)^2+k\)
  • Best for seeing transformations and the turning point.
  • Vertex is \((h,k)\) directly from the equation.
  • Axis of symmetry is \(x=h\). The sign of \(a\) still controls direction.

Case Form C: Factored Form

\(f(x)=a(x-r_1)(x-r_2)\)
  • Best for seeing roots and \(x\)-intercepts immediately.
  • \(x=r_1\) and \(x=r_2\) are the zeros of the function.
  • Roots are \(x\)-intercepts. The vertex lies exactly halfway between them.
3Case 1: \(a>0\), two roots, upward parabola

This is the most common beginner case. If \(a>0\), the parabola opens upward, meaning the vertex is the lowest point on the graph (a global minimum). The function decreases until the vertex, then increases symmetrically.

\(f(x)=x^2-5x+6\)
Factor the expression to reveal the roots:
\(x^2-5x+6=(x-2)(x-3)\)
Find the \(x\)-intercepts by setting \(f(x)=0\). The zero-product property says if a product is zero, at least one factor must be zero:
\(x=2,\qquad x=3\)
Find the \(y\)-intercept by evaluating at \(x=0\). This tells you where the graph starts on the vertical axis:
\(f(0)=6\Rightarrow (0,6)\)
Find the vertex. The parabola is symmetric, so the vertex lies exactly halfway between the roots at \(x=\frac{2+3}{2}=2.5\). Alternatively, use \(x=\frac{-b}{2a}\):
\(x=\frac{-(-5)}{2(1)}=2.5\)
\(f(2.5)=2.5^2-5(2.5)+6=-0.25\)
Vertex: \((2.5,-0.25)\), roots: \(2\) and \(3\), \(y\)-intercept: \((0,6)\)
Because \(a>0\), the graph decreases until the vertex, then increases. The range is bounded below by the vertex \(y\)-value: \(y\ge -0.25\). This minimum value is crucial in optimization problems.
4Case 2: \(a<0\), two roots, downward parabola

If \(a<0\), the parabola opens downward. This flips the entire graph over the \(x\)-axis compared to the positive case. The vertex becomes the highest point (a global maximum), and the range is bounded above instead of below.

\(f(x)=-x^2+5x-6\)
Factor out the negative to simplify factoring:
\(-x^2+5x-6=-(x^2-5x+6)\)
Factor the inside expression normally:
\(-(x-2)(x-3)\)
The roots are still found by setting each factor to zero. The negative sign doesn’t change where the graph crosses the axis:
\(x=2,\qquad x=3\)
Find the vertex using the same symmetry principle. The \(x\)-coordinate is unchanged, but the \(y\)-value flips sign:
\(x=\frac{-5}{2(-1)}=2.5\)
\(f(2.5)=0.25\)
Vertex: \((2.5,0.25)\), maximum value: \(0.25\)
Same roots as the previous example, but the whole graph is reflected over the \(x\)-axis. This is why the vertex changed from a minimum to a maximum, and the range is now \(y\le 0.25\).
5Case 3: Repeated root, parabola touches the \(x\)-axis

Sometimes a quadratic has only one distinct real root. This happens when the parabola just touches the \(x\)-axis at its vertex but does not cross it. Algebraically, the discriminant \(D=b^2-4ac\) equals zero, indicating a double root.

\(f(x)=(x-2)^2\)
  • The root is \(x=2\), but it has multiplicity 2.
  • The vertex is \((2,0)\), sitting exactly on the \(x\)-axis.
  • The graph touches the \(x\)-axis at \((2,0)\) and bounces back up.
  • The discriminant is \(D=0\), confirming a single repeated root.
One real root: \(x=2\), repeated twice
A repeated root means the graph is tangent to the \(x\)-axis. It touches and turns around instead of passing through. This behavior is critical when analyzing inequalities and sign charts.
6Case 4: No real roots, parabola never touches the \(x\)-axis

A quadratic can have no real roots. This means the graph never crosses or touches the \(x\)-axis. The entire parabola floats completely above or below the axis, depending on the sign of \(a\). The discriminant is negative, indicating complex conjugate roots.

\(f(x)=x^2+4\)
  • \(x^2\ge0\) for every real \(x\), so adding 4 guarantees positivity.
  • So \(x^2+4\ge4\). The output never drops below 4.
  • The graph is always above the \(x\)-axis, floating 4 units higher than \(y=x^2\).
  • There are no \(x\)-intercepts because \(f(x)=0\) has no real solution.
No real roots; range: \(y\ge4\)
Use the discriminant to confirm: \(D=b^2-4ac=0^2-4(1)(4)=-16<0\). Since \(D<0\), there are no real roots. The vertex \((0,4)\) is the absolute minimum, and the graph never descends to the axis.
7Case 5: Vertex form and transformations

Vertex form shows transformations clearly. Instead of calculating the vertex, you read it directly. This form is built by shifting, stretching, and reflecting the parent function \(y=x^2\).

\(f(x)=2(x-3)^2-5\)
Vertex is read directly from \((h,k)\):
\((3,-5)\)
Axis of symmetry passes through the vertex:
\(x=3\)
Since \(a=2>0\), it opens upward and is vertically stretched by factor 2, making it narrower than \(y=x^2\).
Range starts at the vertex \(y\)-value and extends upward:
\(y\ge -5\)
Vertex form immediately gives the graph’s position, shape, direction, and range without extra calculation.

Cubic Functions: Complete Case-by-Case Lesson

Change \(a\), \(h\), and \(k\) to graph the transformed cubic \(y=a(x-h)^3+k\). The point \((h,k)\) is the center-like turning-through point called the inflection point.

\(y = a(x-h)^3 + k\)
1What is a cubic function?

A cubic function is a polynomial of degree \(3\). Its highest power is \(x^3\), which fundamentally changes its behavior compared to quadratics. Because the degree is odd, the ends of the graph always point in opposite directions. Cubics can twist, turn, and cross the \(x\)-axis up to three times, making them essential for modeling volume, fluid dynamics, and economic cycles.

\(f(x)=ax^3+bx^2+cx+d,\qquad a \neq 0\)
  • Its end behavior is controlled by the sign of \(a\). Odd degree means one end goes to \(+\infty\) and the other to \(-\infty\).
  • It can have \(1\), \(2\), or \(3\) real \(x\)-intercepts depending on root multiplicity and turning points.
  • It can have no turning point (monotonic), or it can have two turning points (a local max and a local min).
  • It has an inflection point where the curve changes concavity (from cupping up to cupping down, or vice versa).
2Case 1: Basic cubic \(f(x)=x^3\)

The parent cubic function is the simplest cubic. It passes through the origin and has an S-shape. It is an odd function, meaning \(f(-x)=-f(x)\), which creates rotational symmetry about the origin.

\(f(x)=x^3\)

Important points

  • \((-2,-8)\)
  • \((-1,-1)\)
  • \((0,0)\)
  • \((1,1)\)
  • \((2,8)\)

Behavior

  • Left side goes down rapidly as \(x\) becomes more negative.
  • Right side goes up rapidly as \(x\) increases.
  • Domain is all real numbers: \((-\infty,\infty)\).
  • Range is all real numbers: \((-\infty,\infty)\).
\(x^3\) is strictly increasing everywhere, passes through the origin, and has an inflection point at \((0,0)\) where concavity changes.
3Case 2: Negative leading coefficient

If \(a<0\), the cubic is reflected over the \(x\)-axis. The left side goes up and the right side goes down. This reversal happens because multiplying the entire output by a negative flips every \(y\)-value.

\(f(x)=-x^3\)

If \(a>0\)

\(x\to-\infty\Rightarrow f(x)\to-\infty\)

\(x\to+\infty\Rightarrow f(x)\to+\infty\)

If \(a<0\)

\(x\to-\infty\Rightarrow f(x)\to+\infty\)

\(x\to+\infty\Rightarrow f(x)\to-\infty\)

Negative \(a\) reverses the cubic’s end behavior and reflects the entire S-curve across the \(x\)-axis.
4Case 3: Shifted cubic \(a(x-h)^3+k\)

The form \(a(x-h)^3+k\) shows transformations clearly. Instead of recalculating points, you translate the entire parent graph. The inflection point moves from \((0,0)\) to \((h,k)\), and the shape scales by \(|a|\).

\(f(x)=2(x-3)^3-4\)
\(h=3\), so the graph shifts right \(3\) units. Note: the sign inside the parenthesis is opposite to the direction of shift.
\(k=-4\), so the graph shifts down \(4\) units. This moves the entire curve vertically without distortion.
\(a=2\), so the graph is vertically stretched by factor \(2\). It becomes steeper, but the inflection point and symmetry remain intact.
The inflection point is the new center of symmetry:
\((3,-4)\)
\(a(x-h)^3+k\) moves the parent cubic without changing its basic S-shape. The inflection point anchors all transformations.
5Case 4: Cubic with three real roots

A cubic may cross the \(x\)-axis three times. This happens when the graph has two turning points that straddle the axis, creating three distinct zeros. Factoring reveals these intercepts directly.

\(f(x)=x^3-4x\)
Factor common \(x\) to simplify the polynomial:
\(x^3-4x=x(x^2-4)\)
Use difference of squares on the quadratic factor:
\(x(x-2)(x+2)\)
Set each factor equal to zero. The zero-product property applies regardless of degree:
\(x=-2,\quad x=0,\quad x=2\)
\(x\)-intercepts: \((-2,0)\), \((0,0)\), \((2,0)\). The graph crosses the axis at each root because all have odd multiplicity.
6Case 5: Cubic with one real root

Some cubics cross the \(x\)-axis only once. This occurs when the turning points are both above or both below the axis, or when the function is strictly monotonic. The other two roots are complex conjugates.

\(f(x)=x^3+1\)
Set \(f(x)=0\) to find where the output vanishes:
\(x^3+1=0\)
Move \(1\) to the other side to isolate the cubic term:
\(x^3=-1\)
Cube root both sides. Unlike square roots, cube roots preserve sign:
\(x=-1\)
One real \(x\)-intercept: \((-1,0)\). The graph crosses once and continues monotonically.
Cubics always have at least one real root because their ends go to opposite infinities. By the Intermediate Value Theorem, the continuous curve must cross the \(x\)-axis somewhere.

Rational Functions: Complete Case-by-Case Lesson

Change \(a\), \(h\), and \(k\) to graph \(y=\frac{a}{x-h}+k\). This graph has a vertical asymptote \(x=h\) and a horizontal asymptote \(y=k\).

\(y = \frac{a}{x-h} + k\)
1What is a rational function?

A rational function is a ratio of two polynomials. The word “rational” comes from “ratio,” not from being logical. Because division by zero is undefined, any \(x\)-value that makes the denominator zero is excluded from the domain. These exclusions create dramatic graphical features: vertical asymptotes where the function blows up, or holes where a factor cancels out.

\(f(x)=\frac{P(x)}{Q(x)},\qquad Q(x) \neq 0\)
  • The denominator cannot be zero. This restriction defines the domain boundaries.
  • Values that make the denominator zero are excluded from the domain and often create asymptotes.
  • The graph may have vertical asymptotes where the function approaches \(\pm\infty\).
  • The graph may have horizontal or slant asymptotes that describe long-term behavior as \(x\to\pm\infty\).
  • The graph may also have holes if a factor cancels completely, leaving a removable discontinuity.
2Case 1: Basic rational function \(f(x)=\frac{1}{x}\)

This is the parent rational function. Its graph is a rectangular hyperbola with two branches in opposite quadrants. As \(x\) approaches zero, the output explodes. As \(x\) grows large, the output shrinks toward zero.

\(f(x)=\frac{1}{x}\)

Domain

\(x \neq 0\), because division by zero is undefined. The domain splits into two intervals.

\((-\infty,0)\cup(0,\infty)\)

Asymptotes

Vertical asymptote: \(x=0\) (function blows up near zero)

Horizontal asymptote: \(y=0\) (output approaches zero at extremes)

The graph approaches the axes infinitely closely but never touches or crosses them. This boundary behavior defines asymptotes.
3Case 2: Shifted rational function

The form \(\frac{a}{x-h}+k\) shows the asymptotes directly. The entire hyperbola translates without changing shape. The vertical asymptote moves to \(x=h\), and the horizontal asymptote moves to \(y=k\).

\(f(x)=\frac{3}{x-2}+4\)
Vertical asymptote comes from the denominator. Set it to zero to find the forbidden \(x\)-value:
\(x-2=0\Rightarrow x=2\)
Horizontal asymptote comes from the vertical shift. As \(x\to\pm\infty\), the fraction vanishes, leaving only \(k\):
\(y=4\)
The value \(a=3\) controls stretch and branch direction. Positive \(a\) keeps branches in quadrants I/III relative to the asymptote intersection.
Vertical asymptote: \(x=2\), horizontal asymptote: \(y=4\). The center of the hyperbola is \((2,4)\).
4Case 3: Rational function with a vertical asymptote

A vertical asymptote happens when the denominator is zero and the factor does not cancel with the numerator. Near this \(x\)-value, the function grows without bound in opposite directions on either side.

\(f(x)=\frac{x+1}{x-3}\)
Set the denominator equal to zero to find the restriction:
\(x-3=0\Rightarrow x=3\)
Check numerator at \(x=3\). If it’s nonzero, the fraction blows up:
\(3+1=4 \neq 0\)
Since the numerator is not zero, the graph has a vertical asymptote. The function approaches \(+\infty\) on one side and \(-\infty\) on the other.
Vertical asymptote: \(x=3\). The graph splits into two branches separated by this line.
5Case 4: Hole / removable discontinuity

A hole happens when the same factor appears in the numerator and denominator and cancels. The function behaves like the simplified version everywhere except at the canceled value, where it is undefined. This creates a single missing point on the graph.

\(f(x)=\frac{(x-2)(x+1)}{x-2}\)
The denominator is zero at \(x=2\), so \(x=2\) is excluded from the domain.
Cancel the common factor. The function simplifies, but the restriction remains:
\(f(x)=x+1,\qquad x \neq 2\)
Find the missing \(y\)-value using the simplified function. This is where the hole sits:
\(y=2+1=3\)
Hole at \((2,3)\). The graph is a straight line with one missing point.
If a factor cancels, it usually creates a hole. If a factor remains in the denominator after simplifying, it creates a vertical asymptote. Always check multiplicity.
6Case 5: Horizontal asymptote rules

Compare the degree of the numerator with the degree of the denominator. The highest-degree terms dominate as \(x\to\pm\infty\), determining the long-term behavior of the function.

CaseRuleExample
Degree top \(<\) degree bottomHorizontal asymptote is \(y=0\). The denominator grows faster, crushing the fraction to zero.\(\frac{1}{x^2+1}\)
Degree top \(=\) degree bottomUse ratio of leading coefficients. The highest powers cancel, leaving a constant.\(\frac{2x+1}{x-3}\Rightarrow y=2\)
Degree top \(>\) degree bottomNo horizontal asymptote; may have slant asymptote. The numerator grows faster, so the function behaves like a polynomial.\(\frac{x^2+1}{x-1}\)
7Case 6: Finding intercepts of a rational function
\(f(x)=\frac{x-4}{x+2}\)

\(x\)-intercept

Set numerator equal to zero. A fraction is zero only when its numerator is zero (and denominator is not):

\(x-4=0\Rightarrow x=4\)

Check denominator: \(4+2=6 \neq 0\). Valid intercept.

\((4,0)\)

\(y\)-intercept

Set \(x=0\) and evaluate. This gives the starting height on the vertical axis:

\(f(0)=\frac{-4}{2}=-2\)
\((0,-2)\)

Exponential Functions: Complete Case-by-Case Lesson

Change \(a\), \(b\), and \(k\) to graph \(y=a\cdot b^x+k\). The base \(b\) controls growth or decay, while \(k\) moves the horizontal asymptote.

\(y = a\cdot b^x + k\)
1What is an exponential function?

An exponential function is a function where the variable appears in the exponent, not the base. This creates multiplicative change instead of additive change. Each step multiplies the previous output by a constant factor \(b\), leading to rapid acceleration or gradual decay. These functions model population growth, radioactive decay, compound interest, and cooling processes.

\(f(x)=a\cdot b^x,\qquad a \neq 0,\quad b>0,\quad b \neq 1\)
  • \(a\) controls the starting scale and reflection. It is the initial value when \(x=0\).
  • \(b\) controls growth or decay. It is the constant multiplicative factor per unit increase in \(x\).
  • If \(b>1\), the function grows exponentially. The larger \(b\), the faster the explosion.
  • If \(0
  • The parent exponential usually has horizontal asymptote \(y=0\), because \(b^x\) approaches zero but never reaches it.
2Case 1: Exponential growth

If the base is greater than \(1\), the function grows as \(x\) increases. The growth starts slowly, then accelerates dramatically. This compounding effect is why exponential growth eventually outpaces any polynomial.

\(f(x)=2^x\)
\(x\)\(2^x\)Meaning
\(-2\)\(\frac14\)Small positive value, approaching asymptote
\(-1\)\(\frac12\)Still positive, doubling each step right
\(0\)\(1\)Any nonzero base to power \(0\) is \(1\)
\(1\)\(2\)Starts growing, multiplicative pattern begins
\(2\)\(4\)Growth accelerates, each step doubles previous
Growth case: \(b>1\). Domain: all reals. Range: \(y>0\). Horizontal asymptote: \(y=0\).
3Case 2: Exponential decay

If the base is between \(0\) and \(1\), the function decays as \(x\) increases. The output shrinks by a constant fraction each step. It approaches zero asymptotically but never actually reaches it, modeling half-life and depreciation.

\(f(x)=\left(\frac12\right)^x\)
\(x\)\(\left(\frac12\right)^x\)Meaning
\(-2\)\(4\)Large when \(x\) is negative (reciprocal effect)
\(-1\)\(2\)Still above \(1\), halving as \(x\) increases
\(0\)\(1\)Crosses \(y=1\), universal anchor point
\(1\)\(\frac12\)Gets smaller, multiplicative decay begins
\(2\)\(\frac14\)Approaches \(0\), never touches asymptote
Decay case: \(00\).
4Case 3: Vertical shift and horizontal asymptote

Adding \(k\) moves the exponential graph up or down. This shifts the horizontal asymptote from \(y=0\) to \(y=k\). The shape and growth rate stay identical, but the entire curve translates vertically. This models real-world scenarios like temperature approaching room temperature or debt approaching a baseline.

\(f(x)=2^x+3\)
Parent function \(2^x\) has horizontal asymptote:
\(y=0\)
Adding \(+3\) moves everything up \(3\) units. Every output increases by 3.
New horizontal asymptote shifts accordingly:
\(y=3\)
\(y\)-intercept shifts too:
\(f(0)=2^0+3=4\Rightarrow(0,4)\)
Vertical shift changes the asymptote from \(y=0\) to \(y=k\). Range becomes \(y>k\) (if \(a>0\)).
5Case 4: Reflection over the \(x\)-axis

If \(a<0\), the exponential graph reflects over the \(x\)-axis. The parent function’s positive outputs become negative. The curve now approaches the asymptote from below instead of above. This models loss, debt accumulation, or cooling below a reference point.

\(f(x)=-2^x\)
  • The parent \(2^x\) is always positive.
  • The negative sign makes all outputs negative, flipping the curve downward.
  • The graph is below the \(x\)-axis, approaching \(y=0\) from below.
  • The horizontal asymptote is still \(y=0\). Reflection doesn’t move boundaries.
Reflection case: \(a<0\). Range becomes \(y<0\). Asymptote unchanged.
6Case 5: Solving exponential equations without logs

If both sides can be written with the same base, solve by matching exponents. This works because exponential functions are one-to-one: if \(b^m=b^p\) and \(b>0, b\neq1\), then \(m=p\). This method is fast but only works when bases align perfectly.

\(3\cdot2^x=24\)
Divide by \(3\) to isolate the exponential term:
\(2^x=8\)
Rewrite \(8\) as a power of \(2\) to match bases:
\(8=2^3\)
Match exponents. Since bases are identical, exponents must be equal:
\(x=3\)
\(x=3\). Always verify by substituting back into the original equation.
7Case 6: Solving exponential equations with logs

If the number cannot be rewritten with the same base, use logarithms. Logarithms are the inverse of exponentials, allowing you to bring the variable down from the exponent. This method works for any positive base and is essential for real-world modeling.

\(5\cdot3^x=40\)
Divide by \(5\) to isolate the exponential:
\(3^x=8\)
Take logarithm of both sides. Any base works; common or natural log is standard:
\(\log(3^x)=\log(8)\)
Bring down the exponent using the power rule \(\log(b^x)=x\log b\):
\(x\log3=\log8\)
Solve for \(x\) by dividing:
\(x=\frac{\log8}{\log3}\)
\(x=\frac{\log8}{\log3}\approx1.89\). This exact form is preferred; decimal is for interpretation.
