Linear Functions — Complete Lesson

A linear function is one of the most important ideas in algebra. It describes a relationship where one quantity changes at a constant rate compared with another quantity. Its graph is always a straight line.

\(f(x)=mx+b\quad \text{or}\quad y=mx+b\)
  • \(x\) is the input, also called the independent variable.
  • \(y\) or \(f(x)\) is the output, also called the dependent variable.
  • \(m\) is the slope, which tells how fast the line rises or falls.
  • \(b\) is the \(y\)-intercept, which tells where the line crosses the \(y\)-axis.
Big idea: A linear function has a constant rate of change. That means every time \(x\) increases by the same amount, \(y\) also changes by the same amount.
Function form

\(f(x)=2x+3\) means the output depends on the input \(x\).

Graph form

\(y=2x+3\) means the same thing, but it is often used when drawing the line.

Interactive Linear Function Graph

Change \(m\) and \(b\). Watch how the line moves. The slider \(m\) controls steepness and direction, while \(b\) moves the line up or down.

\(y = mx + b\)

1Understanding Slope

The slope measures the rate of change. In simple words, it tells how much \(y\) changes when \(x\) increases by \(1\).

\(m=\dfrac{\text{change in }y}{\text{change in }x}=\dfrac{\text{rise}}{\text{run}}\)

\(m\gt0\)

The line rises from left to right.

\(m\lt0\)

The line falls from left to right.

\(m=0\)

The line is horizontal.

Example: Positive Slope

\(y=2x+1\)

The slope is \(2\). This means when \(x\) increases by \(1\), \(y\) increases by \(2\). The line goes upward.

Example: Negative Slope

\(y=-3x+4\)

The slope is \(-3\). This means when \(x\) increases by \(1\), \(y\) decreases by \(3\). The line goes downward.

Example: Zero Slope

\(y=5\)

The slope is \(0\). The output is always \(5\), so the graph is a horizontal line.

2Understanding the \(y\)-Intercept

The \(y\)-intercept is the point where the line crosses the \(y\)-axis. On the \(y\)-axis, \(x=0\). So to find the \(y\)-intercept, substitute \(x=0\).

\(y=mx+b\)

When \(x=0\):

\(y=m(0)+b=b\)
Important: The \(y\)-intercept is always \((0,b)\) in slope-intercept form.

Example

\(y=3x-4\)

Here, \(b=-4\). So the line crosses the \(y\)-axis at:

✅\((0,-4)\)

3How to Graph a Linear Function

The easiest way to graph \(y=mx+b\) is to use the \(y\)-intercept and the slope.

Example

\(y=\dfrac{2}{3}x+1\)
Identify the \(y\)-intercept: \(b=1\). Plot the point \((0,1)\).
Identify the slope: \(m=\dfrac{2}{3}\). This means rise \(2\), run \(3\).
From \((0,1)\), move \(2\) units up and \(3\) units right. This gives another point: \((3,3)\).
Draw a straight line through the two points.
You only need two points to draw a line, but the slope must stay the same everywhere on the line.

4Finding \(y\) When \(x\) Is Given

If you know \(x\), substitute it into the function and calculate \(y\).

Example

\(y=3x+2\)

Find \(y\) when \(x=4\).

Substitute \(x=4\):
\(y=3(4)+2\)
Multiply first:
\(y=12+2\)
Add:
\(y=14\)
✅Final Answer: \(y=14\)

5Finding \(x\) When \(y\) Is Given

Sometimes you are given the output \(y\), and you must find the input \(x\). Then you solve a linear equation.

Example

\(y=2x+5\)

Find \(x\) when \(y=17\).

Replace \(y\) with \(17\):
\(17=2x+5\)
Subtract \(5\) from both sides:
\(12=2x\)
Divide by \(2\):
\(x=6\)
✅Final Answer: \(x=6\)

6Finding Intercepts

Intercepts are the points where a line crosses the coordinate axes.

\(y\)-intercept

Set \(x=0\).

\(y=2(0)+6=6\)

So the \(y\)-intercept is \((0,6)\).

\(x\)-intercept

Set \(y=0\).

\(0=2x+6\Rightarrow x=-3\)

So the \(x\)-intercept is \((-3,0)\).

Full Example

\(y=-4x+8\)
  • For the \(y\)-intercept, set \(x=0\): \(y=8\), so the point is \((0,8)\).
  • For the \(x\)-intercept, set \(y=0\): \(0=-4x+8\), so \(x=2\), and the point is \((2,0)\).

7Finding Slope from Two Points

If two points are given, use the slope formula.

\(m=\dfrac{y_2-y_1}{x_2-x_1}\)

Example

Find the slope through \((1,3)\) and \((4,9)\).

Label the points:
\((x_1,y_1)=(1,3),\quad (x_2,y_2)=(4,9)\)
Substitute into the formula:
\(m=\dfrac{9-3}{4-1}\)
Simplify:
\(m=\dfrac{6}{3}=2\)
✅Slope: \(m=2\)
If the denominator \(x_2-x_1\) is zero, the line is vertical. A vertical line is not a function because one \(x\)-value has many \(y\)-values.

8Writing the Equation of a Line

Case 1: Slope and \(y\)-intercept are given

If \(m=3\) and \(b=-2\), put them into \(y=mx+b\).

\(y=3x-2\)

Case 2: Slope and one point are given

Suppose the slope is \(m=2\), and the line passes through \((3,7)\). Start with:

\(y=mx+b\)
Substitute \(m=2\):
\(y=2x+b\)
Use the point \((3,7)\). This means \(x=3\), \(y=7\):
\(7=2(3)+b\)
Solve for \(b\):
\(7=6+b\Rightarrow b=1\)
✅Equation: \(y=2x+1\)

Case 3: Two points are given

Find the equation of the line through \((1,2)\) and \((3,6)\).

First find the slope:
\(m=\dfrac{6-2}{3-1}=\dfrac{4}{2}=2\)
Use \(y=mx+b\):
\(y=2x+b\)
Substitute point \((1,2)\):
\(2=2(1)+b\Rightarrow b=0\)
✅Equation: \(y=2x\)

9Special Lines

Horizontal line
\(y=c\)

Slope is \(0\). Example: \(y=4\). It is a function.

Vertical line
\(x=c\)

Slope is undefined. Example: \(x=4\). It is not a function.

A vertical line fails the vertical line test, because one input \(x\) can have many outputs \(y\).

10Parallel and Perpendicular Lines

Parallel Lines

Parallel lines have the same slope but different \(y\)-intercepts.

\(y=2x+1\quad \text{and}\quad y=2x-5\)

Both slopes are \(2\), so the lines are parallel.

Perpendicular Lines

Perpendicular lines have slopes that are negative reciprocals. Their slopes multiply to \(-1\).

\(m_1\cdot m_2=-1\)

Example:

\(m_1=2,\quad m_2=-\dfrac{1}{2}\)

Since \(2\cdot\left(-\dfrac{1}{2}\right)=-1\), the lines are perpendicular.

11Real-Life Linear Models

Linear functions are used when something changes by a constant amount.

Example: Taxi Cost

A taxi charges \(5\) dollars as a starting fee and \(2\) dollars per kilometer.

\(C=2x+5\)
  • \(x\) = number of kilometers
  • \(2\) = cost per kilometer
  • \(5\) = starting fee
  • \(C\) = total cost

If you travel \(8\) kilometers:

\(C=2(8)+5=21\)
✅Total cost: \(21\) dollars

12Common Mistakes

1. Mixing up slope and intercept

In \(y=mx+b\), \(m\) controls steepness, while \(b\) controls where the line crosses the \(y\)-axis.

2. Forgetting that \(b\) is a point, not just a number

If \(b=5\), the \(y\)-intercept is \((0,5)\), not just \(5\).

3. Using the slope formula in the wrong order

If you start with \(y_2-y_1\), you must also use \(x_2-x_1\). Do not switch the order halfway.

\(m=\dfrac{y_2-y_1}{x_2-x_1}\quad \text{or}\quad m=\dfrac{y_1-y_2}{x_1-x_2}\)

Both are correct, but mixing them is wrong.

4. Thinking every straight line is a function

Vertical lines are straight, but they are not functions because one \(x\)-value has many \(y\)-values.

13Final Summary

  1. A linear function has the form \(y=mx+b\).
  2. The graph of a linear function is a straight line.
  3. \(m\) is the slope, or rate of change.
  4. \(b\) is the \(y\)-intercept.
  5. Positive slope means the line rises.
  6. Negative slope means the line falls.
  7. Zero slope means the line is horizontal.
  8. To find the \(y\)-intercept, set \(x=0\).
  9. To find the \(x\)-intercept, set \(y=0\).
  10. To find slope from two points, use \(m=\dfrac{y_2-y_1}{x_2-x_1}\).
