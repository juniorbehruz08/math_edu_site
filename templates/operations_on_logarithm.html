Operations on Logarithms — Complete Lesson

Operations on logarithms are rules that help us combine, separate, and simplify logarithmic expressions. These rules come from exponent laws because logarithms and exponents are inverse ideas.

\(\log_b a=c \quad \Longleftrightarrow \quad b^c=a\)

In this lesson, we focus on the product rule, quotient rule, power rule, and change of base formula.

Main idea: Logarithm operations are shortcuts for working with multiplication, division, and powers inside logarithms.

1Before Using Logarithm Rules

Logarithm rules work only when the logarithms are valid and the bases match.

Same base

Most combining rules require the same base.

\(\log_b x+\log_b y\)
Positive arguments

The values inside the logarithms must be positive.

\(x>0,\quad y>0\)
Valid base

The base must be positive and not equal to one.

\(b>0,\quad b\ne1\)
No sum rule

There is no simple rule for \(\log_b(x+y)\).

2Product Rule

The product rule says that the logarithm of a product can be written as the sum of two logarithms.

\(\log_b(xy)=\log_b x+\log_b y\)

This rule works in both directions. You can expand one logarithm into two, or combine two logarithms into one.

Example 1: Expanding

\(\log_2(8x)=\log_2 8+\log_2 x\)

Since \(\log_2 8=3\), we get:

✓\(\log_2(8x)=3+\log_2x\)

Example 2: Combining

\(\log_3 x+\log_3 5=\log_3(5x)\)
Remember: Addition outside logarithms becomes multiplication inside one logarithm.

3Quotient Rule

The quotient rule says that the logarithm of a quotient can be written as the difference of two logarithms.

\(\log_b\left(\frac{x}{y}\right)=\log_b x-\log_b y\)

This rule also works in both directions.

Example 1: Expanding

\(\log_5\left(\frac{x}{25}\right)=\log_5x-\log_5 25\)

Since \(\log_5 25=2\), we get:

✓\(\log_5\left(\frac{x}{25}\right)=\log_5x-2\)

Example 2: Combining

\(\log_4 x-\log_4 y=\log_4\left(\frac{x}{y}\right)\)
Remember: Subtraction outside logarithms becomes division inside one logarithm.

4Power Rule

The power rule says that an exponent inside a logarithm can move to the front.

\(\log_b(x^n)=n\log_bx\)

Example 1: Expanding

\(\log_2(x^5)=5\log_2x\)

Example 2: Combining

\(3\log_7x=\log_7(x^3)\)
Important: The power rule works only when the exponent belongs to the argument of the logarithm.

5Expanding Logarithmic Expressions

Expanding means rewriting one logarithm as several simpler logarithms.

\(\log_2(8x^3y)\)
Use the product rule: \(\log_2 8+\log_2(x^3)+\log_2y\).
Use the power rule: \(\log_2 8+3\log_2x+\log_2y\).
Evaluate \(\log_2 8=3\).
✓Final Answer: \(3+3\log_2x+\log_2y\)

6Condensing Logarithmic Expressions

Condensing means rewriting several logarithms as one logarithm.

\(2\log_3x+\log_3y-\log_3z\)
Move the coefficient using the power rule: \(\log_3(x^2)+\log_3y-\log_3z\).
Combine addition using multiplication: \(\log_3(x^2y)-\log_3z\).
Combine subtraction using division.
✓Final Answer: \(\log_3\left(\frac{x^2y}{z}\right)\)

7Change of Base Formula

The change of base formula lets us rewrite a logarithm using another base. This is useful for calculators.

\(\log_ba=\frac{\log a}{\log b}\)

It can also be written with natural logarithms:

\(\log_ba=\frac{\ln a}{\ln b}\)

Example

\(\log_2 7=\frac{\log 7}{\log 2}\)
Meaning: We changed a base \(2\) logarithm into base \(10\) logarithms.

8Reciprocal Logarithm Rule

Another useful rule comes directly from the change of base formula. When a logarithm is in the denominator, the base and the argument can switch positions.

\(\frac{1}{\log_b a}=\log_a b\)

This works because of the change of base formula:

\(\log_b a=\frac{\log a}{\log b}\)

If we take the reciprocal, the fraction flips:

\(\frac{1}{\log_b a}=\frac{\log b}{\log a}=\log_a b\)

Example

\(\frac{1}{\log_2 8}=\log_8 2\)

Since \(\log_2 8=3\), the left side is \(\frac{1}{3}\). Also, \(\log_8 2=\frac{1}{3}\), because \(8^{1/3}=2\).

✓So, \(\frac{1}{\log_2 8}=\log_8 2=\frac{1}{3}\)
Memory trick: If the logarithm is under \(1\), switch the base and the argument.

9Common Mistakes

1. Using a fake addition rule

\(\log_b(x+y)\) is not equal to \(\log_bx+\log_by\).

2. Using a fake subtraction rule

\(\log_b(x-y)\) is not equal to \(\log_bx-\log_by\).

3. Combining logarithms with different bases

You cannot combine \(\log_2x+\log_3y\) using the product rule because the bases are different.

4. Forgetting domain restrictions

Every logarithm argument must be positive. Always check that the expression inside each logarithm is greater than zero.

10Final Summary

Rule Formula Meaning
Product Rule \(\log_b(xy)=\log_bx+\log_by\) Multiplication becomes addition.
Quotient Rule \(\log_b\left(\frac{x}{y}\right)=\log_bx-\log_by\) Division becomes subtraction.
Power Rule \(\log_b(x^n)=n\log_bx\) An exponent can move to the front.
Change of Base \(\log_ba=\frac{\log a}{\log b}\) A logarithm can be rewritten using another base.
Reciprocal Rule \(\frac{1}{\log_b a}=\log_a b\) The base and argument switch positions.
