Inverse Trigonometric Functions — Modular Guide

Inverse trigonometric functions reverse the basic trigonometric functions. They answer questions like: “Which angle has this sine value?” or “Which angle has this tangent value?”

Because ordinary trigonometric functions repeat, inverse trigonometric functions must use restricted output ranges. Those ranges make each inverse function give one principal answer.

Main idea: Inverse trigonometric functions take a ratio or coordinate value and return an angle.

1What Inverse Trigonometric Functions Mean

A normal trigonometric function starts with an angle and gives a value. An inverse trigonometric function starts with a value and gives an angle.

Normal trig function Inverse trig function Meaning
\(\sin\theta=x\) \(\arcsin x=\theta\) The angle whose sine is \(x\).
\(\cos\theta=x\) \(\arccos x=\theta\) The angle whose cosine is \(x\).
\(\tan\theta=x\) \(\arctan x=\theta\) The angle whose tangent is \(x\).
\(\cot\theta=x\) \(\operatorname{arccot}x=\theta\) The angle whose cotangent is \(x\).
Important: The answer of an inverse trigonometric function is an angle, not a ratio.

2Why We Restrict the Range

Trigonometric functions repeat. For example, many different angles can have the same sine value. To make an inverse function, we choose one main interval where each output is used only once.

\(\arcsin x\)

Returns angles from \(-90^\circ\) to \(90^\circ\).

\(\arccos x\)

Returns angles from \(0^\circ\) to \(180^\circ\).

\(\arctan x\)

Returns angles between \(-90^\circ\) and \(90^\circ\).

\(\operatorname{arccot}x\)

Usually returns angles from \(0^\circ\) to \(180^\circ\).

1First, remember what \(\sin(x)\) means

For a right triangle:

\(\sin(\theta)=\frac{\text{opposite}}{\text{hypotenuse}}\)

Example:

\(\sin(30^\circ)=\frac12\)

This means:

“The sine of \(30^\circ\) equals \(1/2\).”

2Then what is \(\arcsin(x)\)?

\(\arcsin(x)\)

asks the reverse question:

“Which angle has sine equal to \(x\)?”

So:

\(\arcsin\left(\frac12\right)=30^\circ\)

because

\(\sin(30^\circ)=\frac12\)

3Very important idea: inverse ≠ reciprocal

Many students confuse these:

\(\arcsin(x)\ne\frac{1}{\sin(x)}\)

The reciprocal of sine is:

\(\csc(x)\)

while inverse sine means:

\(\sin^{-1}(x)=\arcsin(x)\)

which means:

“the angle whose sine is \(x\).”

4Why do we even need arcsin?

Sometimes we know the ratio but do not know the angle.

Example:

\(\sin(\theta)=0.8\)

To find the angle:

\(\theta=\arcsin(0.8)\)

Calculator:

\(\theta\approx53.13^\circ\)

5Important restriction (VERY IMPORTANT)

Sine is not one-to-one on all real numbers.

For example:

\(\sin(30^\circ)=\sin(150^\circ)=\frac12\)

So if we simply ask:

\(\arcsin\left(\frac12\right)\)

which angle should we choose?

Option 1\(30^\circ\)
Option 2\(150^\circ\)
Option 3\(390^\circ\)
To avoid chaos, mathematics chooses a special interval.

6Principal Value of Arcsin

For \(\arcsin(x)\), the output angle is restricted to:

\([-90^\circ,90^\circ]\)

or in radians:

\(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\)

This interval is called the principal branch.

So:

\(\arcsin\left(\frac12\right)=30^\circ\)

NOT \(150^\circ\).

7Domain and Range

Domain

The input of arcsin must satisfy:

\(-1\le x\le1\)

because sine values can never exceed \(1\) or go below \(-1\).

So:

✓ possible:
\(\arcsin(0.2)\)
✓ possible:
\(\arcsin(-1)\)
× impossible:
\(\arcsin(2)\)

Range

\(-\frac{\pi}{2}\le y\le\frac{\pi}{2}\)

or

\(-90^\circ\le y\le90^\circ\)

Useful symmetry rule

Arcsine is an odd function, so a negative input gives a negative output:

\(\arcsin(-x)=-\arcsin(x)\)

Example:

\(\arcsin\left(-\frac12\right)=-\arcsin\left(\frac12\right)=-30^\circ\)

8Graph intuition

The graph of:

\(y=\arcsin(x)\)

is the reflection of the restricted sine graph across:

\(y=x\)

because inverse functions swap:

  • inputs
  • outputs

8Five Common Problem Types in Inverse Trigonometry

In inverse trigonometric functions, most mistakes happen because students try to memorize answers without checking the domain, the principal range, and the order of the functions. This section shows the most common problem types and how to solve them step by step.

Core habit: Always ask two questions: “Is the input allowed?” and “Is the final angle inside the principal range?”

Problem Type 1: Evaluating Basic Inverse Trig Values

These problems ask you to find an angle from a trigonometric value.

\(\arcsin\left(\frac12\right)\)
First identify the function: \(\arcsin\) means “the angle whose sine is the input.”
Check the input: \(\frac12\) is allowed because it is between \(-1\) and \(1\).
Use the principal range of \(\arcsin\): \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\).
Find the angle in that range whose sine is \(\frac12\): that angle is \(\frac{\pi}{6}\).
✓Final answer: \(\arcsin\left(\frac12\right)=\frac{\pi}{6}\)
Nuance: \(\arcsin\left(\frac12\right)\) is not asking for every angle with sine \(\frac12\). It asks for the principal angle only.

Problem Type 2: Checking Whether the Input Is Allowed

Before solving, check the domain. Some inverse trig functions do not accept every input.

\(\arccos(2)\)
The function is \(\arccos x\).
The domain of \(\arccos x\) is \([-1,1]\).
The input \(2\) is outside this domain.
✓Final answer: \(\arccos(2)\) is undefined in real numbers.
Nuance: \(\arcsin x\) and \(\arccos x\) require \(-1\le x\le1\). But \(\arctan x\) and \(\operatorname{arccot}x\) accept all real inputs.

Problem Type 3: Trig Outside Inverse Trig

If the outside trig function matches the inside inverse trig function, the expression usually simplifies directly.

\(\cos(\arccos x)\)
Let \(\theta=\arccos x\).
Then \(\cos\theta=x\).
So \(\cos(\arccos x)=x\).
But remember the input restriction: \(x\) must be between \(-1\) and \(1\).
✓Final answer: \(\cos(\arccos x)=x,\ -1\le x\le1\)
Nuance: This type is usually direct because the outside function cancels the inside inverse function on its valid domain.

Problem Type 4: Inverse Trig Outside Trig

This is the tricky case you mentioned. \(\arccos(\cos x)\) is not always \(x\), because \(\arccos\) only returns angles from \(0\) to \(\pi\).

\(\arccos\left(\cos\frac{5\pi}{3}\right)\)
First find the cosine value: \(\cos\frac{5\pi}{3}=\frac12\).
Now solve \(\arccos\left(\frac12\right)\).
The answer must be inside the principal range of \(\arccos\): \([0,\pi]\).
The angle in \([0,\pi]\) whose cosine is \(\frac12\) is \(\frac{\pi}{3}\).
✓Final answer: \(\arccos\left(\cos\frac{5\pi}{3}\right)=\frac{\pi}{3}\), not \(\frac{5\pi}{3}\)
Expression When it equals \(x\) Nuance
\(\arcsin(\sin x)\) \(-\frac{\pi}{2}\le x\le\frac{\pi}{2}\) The answer must be in the arcsin principal range.
\(\arccos(\cos x)\) \(0\le x\le\pi\) The answer must be in the arccos principal range.
\(\arctan(\tan x)\) \(-\frac{\pi}{2}<x<\frac{\pi}{2}\) The answer must be in the arctan principal range.
\(\operatorname{arccot}(\cot x)\) \(0<x<\pi\) The answer must be in the arccot principal range.

Problem Type 5: Simplifying Expressions Like \(\sin(\arccos x)\)

These problems do not cancel directly because the outside function and inside inverse function do not match. The best method is to draw a right triangle or use \(\sin^2\theta+\cos^2\theta=1\).

\(\sin(\arccos x)\)
Let \(\theta=\arccos x\). Then \(\cos\theta=x\).
Since \(\arccos x\) returns \(\theta\in[0,\pi]\), the sine of \(\theta\) is never negative.
Use \(\sin^2\theta+\cos^2\theta=1\).
So \(\sin^2\theta=1-x^2\), and because \(\sin\theta\ge0\), \(\sin\theta=\sqrt{1-x^2}\).
✓Final answer: \(\sin(\arccos x)=\sqrt{1-x^2}\)
Nuance: Do not write \(\pm\sqrt{1-x^2}\). Because \(\arccos x\) returns angles from \(0\) to \(\pi\), sine is nonnegative there.

Problem Type 6: Solving Equations with Inverse Trig

When an inverse trig expression equals an angle, rewrite it as a normal trig statement.

\(\arcsin(2x-1)=\frac{\pi}{6}\)
Rewrite the equation: if \(\arcsin(2x-1)=\frac{\pi}{6}\), then \(\sin\frac{\pi}{6}=2x-1\).
Use the known value: \(\sin\frac{\pi}{6}=\frac12\).
So \(2x-1=\frac12\).
Solve: \(2x=\frac32\), so \(x=\frac34\).
Check the input: \(2x-1=\frac12\), which is inside \([-1,1]\). So the answer is valid.
✓Final answer: \(x=\frac34\)
Nuance: Always check the domain after solving. Algebra can produce answers that make the inverse trig input invalid.

Problem Type 7: Solving Equations Where the Answer Must Be Principal

Sometimes the expression looks like it should give many angles, but inverse trig gives only the principal angle.

\(\arctan x=\frac{3\pi}{4}\)
Check the principal range of \(\arctan x\): \(\left(-\frac{\pi}{2},\frac{\pi}{2}\right)\).
The angle \(\frac{3\pi}{4}\) is outside this range.
Therefore, no real \(x\) can make \(\arctan x=\frac{3\pi}{4}\).
✓Final answer: no real solution.
Final memory: For inverse trig problems, solving is not just calculation. You must check domain, principal range, and whether the composition order changes the answer.

10Final Summary

Function Domain Principal range Meaning
\(\arcsin x\) \([-1,1]\) \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\) Angle whose sine is \(x\).
\(\arccos x\) \([-1,1]\) \([0,\pi]\) Angle whose cosine is \(x\).
\(\arctan x\) \((-\infty,\infty)\) \(\left(-\frac{\pi}{2},\frac{\pi}{2}\right)\) Angle whose tangent is \(x\).
\(\operatorname{arccot}x\) \((-\infty,\infty)\) \((0,\pi)\) Angle whose cotangent is \(x\).

Common Mistakes

Forgetting the principal range

Inverse trig functions return the principal angle, not every possible angle.

Using invalid inputs

\(\arcsin x\) and \(\arccos x\) do not accept inputs less than \(-1\) or greater than \(1\).

Pulling out negatives incorrectly

\(\arccos(-x)\) is not equal to \(-\arccos x\).

Confusing inverse with reciprocal

\(\arcsin x\) is not the same as \(\frac{1}{\sin x}\).
