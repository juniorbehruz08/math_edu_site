Linear Equation with One Variable

A linear equation with one variable has the form:

ax+b=cax + b = 0

where:

  • aa, b and cb are given numbers (constants),
  • xx is the unknown variable.
  • When solving a linear equation, known terms are moved to one side, and unknown terms are moved to the other side.

  • If we want to find xx in the equation 2x+5=3y−52x + 5 = 3y - 5 , we consider y as a known value. The solution is:

    x=3y−102x = \frac{3y - 10}{2}​

Moving Terms Across the Equation (Transposition) with All Operations

When solving linear equations, we move terms to the other side by changing their operation. Here’s how it works for addition, subtraction, multiplication, and division:

1. Addition to Subtraction

If a term is added, move it by subtracting.

Example:

x+7=12x + 7 = 12

Move +7 to the other side:

x=12−7x = 12 - 7 x=5x = 5

2. Subtraction to Addition

If a term is subtracted, move it by adding.

Example:

x−4=9x - 4 = 9

Move−4 to the other side:

x=9+4x = 9 + 4 x=13x = 13

3. Multiplication to Division

If a variable is multiplied, move the number by dividing.

Example:

5x=205x = 20

Move \times 5×5 to the other side by dividing:

x=205x = \frac{20}{5}​ x=4x = 4

4. Division to Multiplication

If a variable is divided, move the number by multiplying.

Example:

x3=7\frac{x}{3} = 7​

Move \div 3÷3 to the other side by multiplying:

x=7×3x = 7 \times 3 x=21x = 21

Example with All Operations

Solve:

2x+53=7\frac{2x + 5}{3} = 7​

Step 1: Move \div 3÷3 to the other side by multiplying:

2x+5=7×32x + 5 = 7 \times 3 2x+5=212x + 5 = 21

Step 2: Move +5+5 by subtracting:

2x=21−52x = 21 - 5 2x=162x = 16

Step 3: Move \times 2×2 by dividing:

x=162x = \frac{16}{2}​ x=8x = 8

Key Rules Recap:

  • Addition → Subtract when moving
  • Subtraction → Add when moving
  • Multiplication → Divide when moving
  • Division → Multiply when moving
  • If a linear equation simplifies to the form x=x, then x has infinitely many solutions.
  • If, when simplifying the equation, a false equality remains (for example, 8=3), then the equation has no solution.

Quadratic Equations

A quadratic equation is a polynomial equation of the form:

ax2+bx+c=0ax^2 + bx + c = 0

where:

  • a, bb, and cc are constants,
  • x1 + x2 = -b and x1 * x2 = c where a = 1
  • x1 + x2 = -b/a and x1 + x2 = c/a where a ≠ 1
  • a≠0a \neq 0 (if a=0a = 0, the equation becomes linear).

To solve quadratic equations, we can use methods such as:

  1. Factoring
  2. Quadratic formula:
x=−b±b2−4ac2ax = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} ​

Examples

Example 1 (Factoring Method)

Solve:

x2−5x+6=0x^2 - 5x + 6 = 0

Step 1: Find two numbers that multiply to 6 (c) and add up to -5 (b). These numbers are -2 and -3.

Step 2: Factor the equation:

(x−2)(x−3)=0(x - 2)(x - 3) = 0

Step 3: Solve for xxx:

x−2=0⇒x=2x - 2 = 0 \quad \Rightarrow \quad x = 2 x−3=0⇒x=3x - 3 = 0 \quad \Rightarrow \quad x = 3

Solution: x=2,x = 3x=3

Example 2 (Quadratic Formula)

Solve:

  • x1 + x2 = -b and x1 * x2 = c where a = 1
  • x1 + x2 = -b/a and x1 + x2 = c/a where a ≠ 1
    • D = b2−4ac
  • if D > 0, equation has 2 solutions. If D = 0, equation has one solution. If D < 0, equation has not a solution
2x2−3x−5=02x^2 - 3x - 5 = 0

Here, a=2a = 2, b=−3b = -3, c=−5c = -5.
Using the quadratic formula:

x=−(−3)±(−3)2−4(2)(−5)2(2)x = \frac{-(-3) \pm \sqrt{(-3)^2 - 4(2)(-5)}}{2(2)} ​​ x=3±9+404x = \frac{3 \pm \sqrt{9 + 40}}{4} ​​ x=3±494x = \frac{3 \pm \sqrt{49}}{4} ​​ x=3±74x = \frac{3 \pm 7}{4}​

Solving for both cases:

x=3+74=104=2.5x = \frac{3 + 7}{4} = \frac{10}{4} = 2.5​​ x=3−74=−44=−1x = \frac{3 - 7}{4} = \frac{-4}{4} = -1​​

Solution: x=2.5, x−1x = 2.5, -1

Linear Inequalities Explanation

A linear inequality is similar to a linear equation, but instead of an equal sign (=)(=), it uses inequality symbols:

  • >> (greater than)
  • << (less than)
  • \geq≥ (greater than or equal to)
  • \leq≤ (less than or equal to)

Solving Linear Inequalities

Solving a linear inequality follows the same steps as solving a linear equation, but with one important rule:

🔹 When multiplying or dividing by a negative number, flip the inequality sign.

Case 1: Simple Inequality

Solve:

2x+3>72x + 3 > 7

Steps:

  1. Subtract 3 from both sides: 2x>42x > 4
  2. Divide by 2: x>2x > 2

✅ Solution: x>2x > 2 (which means xx can be any number greater than 2).

Case 2: Inequality with Negative Coefficient

Solve:

−3x+5≤2-3x + 5 \leq 2

Steps:

  1. Subtract 5 from both sides: −3x≤−3-3x \leq -3
  2. Divide by -3 (and flip the sign): x≥1x \geq 1

✅ Solution: x≥1x \geq 1 (which means xx can be 1 or any number greater than 1).

Case 3: Inequality That is Always True

Solve:

2x+5>2x+32x + 5 > 2x + 3

Steps:

  1. Subtract 2x2x from both sides: 5>35 > 3

Since 5>35 > 3 is always true, the inequality holds for all real numbers.

✅ Solution: All real numbers (R)(\mathbb{R}).

Case 4: Inequality with No Solution

Solve:

3x+4<3x−23x + 4 < 3x - 2

Steps:

  1. Subtract 3x3x from both sides: 4<−24 < -2

Since 4<−24 < -2 is false, the inequality has no solution.

❌ Solution: No solution (∅)(\emptyset).

Quadratic Inequalities Explanation

A quadratic inequality is similar to a quadratic equation, but instead of an equal sign (=)(=), it uses inequality symbols:

  • ax2+bx+c>0ax^2 + bx + c > 0 (greater than)
  • ax2+bx+c<0ax^2 + bx + c < 0 (less than)
  • ax2+bx+c≥0ax^2 + bx + c \geq 0 (greater than or equal to)
  • ax2+bx+c≤0ax^2 + bx + c \leq 0 (less than or equal to)

Steps to Solve Quadratic Inequalities

  1. Rearrange the inequality so that one side is 0.
  2. Solve the corresponding quadratic equation ax2+bx+c=0ax^2 + bx + c = 0 to find critical points (roots).
  3. Determine the sign of each interval by testing values in the factored form.
  4. Choose the correct intervals based on the inequality symbol.

Case 1: Standard Quadratic Inequality

Solve:

x2−5x+6>0x^2 - 5x + 6 > 0

Steps:

  1. Solve the quadratic equation x2−5x+6=0x^2 - 5x + 6 = 0

    (x−2)(x−3)=0(x - 2)(x - 3) = 0

    Roots: x=2,x=3x = 2, x = 3

  2. Find sign intervals:

    • Pick test points: x<2x < 2 (e.g. x=1x = 1), 2<x<32 < x < 3 (e.g. x=2.5x = 2.5), x>3x > 3 (e.g. x=4x = 4).
    • We check the first case. We replace 𝑥 with a number less than 2 and simplify, for example, 1. We get a positive result. Therefore, the interval for this case is (−∞;2) since x<2.
    • We repeat it again for the second case. We replace x with a number greater than 2 and less than 3, for example, 2.5. We get a negative result. Therefore, the interval for this case is (2;3) since 2 < x < 3
    • We repeat it for the third case. We replace x with a number greater than 3, for example, 4. We get a positive result. Therefore, the interval for this case is (3;∞) since x > 3

  3. Select positive intervals:
    Since we need >0> 0, the solution is:

    (−∞,2)∪(3,∞)(-\infty, 2) \cup (3, \infty)

✅ Solution: x<2x < 2 and x>3x > 3.

Case 2: Less Than or Equal to (Includes Roots)

Solve:

x2−4x+3≤0x^2 - 4x + 3 \leq 0

Steps:

  1. Solve x2−4x+3=0x^2 - 4x + 3 = 0

    (x−1)(x−3)=0(x - 1)(x - 3) = 0

    Roots: x=1,x=3x = 1, x = 3

  2. Test sign intervals:

    We check three cases: x < 1, 1 < x < 3 and x > 3

    • We check the first case. We replace 𝑥 with a number less than 1 and simplify, for example, 0. We get a positive result. Therefore, the interval for this case is (−∞;1] since x<1.
    • We repeat it again for the second case. We replace x with a number greater than 1 and less than 3, for example, 2. We get a negative result. Therefore, the interval for this case is [1;3] since 1 < x < 3
    • We repeat it for the third case. We replace x with a number greater than 3, for example, 4. We get a positive result. Therefore, the interval for this case is [3;∞) since x > 3

  3. Choose ≤0\leq 0 (negative region, including roots):

    [1,3][1, 3]

✅ Solution: 1≤x≤31 \leq x \leq 3.

Case 3: No Solution (Always False Inequality)

Solve:

x2+1<0x^2 + 1 < 0
  1. x2+1=0x^2 + 1 = 0 has no real roots because x2+1≥1x^2 + 1 \geq 1for all real xx.
  2. Since x2+1x^2 + 1 is always positive, it can never be less than 0.

❌ Solution: No solution (∅)(\emptyset).

Case 4: Always True (All Real Numbers as Solution)

Solve:

x2+1≥0x^2 + 1 \geq 0
  1. x2+1x^2 + 1 is always positive for all x, since x2x^2 is never negative.
  2. Since the inequality is ≥0\geq 0, it is always true.

✅ Solution: All real numbers (R)(\mathbb{R}).
