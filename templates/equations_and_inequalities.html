Linear Equation with One Variable

A linear equation with one variable has the form:

ax+b=c

where:

  • a, b and c are constants (known numbers)
  • x is the unknown variable
  • When solving, move known terms to one side and unknown terms to the other
  • Example: In 2x + 5 = 3y − 5, treating y as known: x=3y−102

Moving Terms Across the Equation (Transposition)

1. Addition → Subtraction

If a term is added, move it by subtracting.

Example: x + 7 = 12 → x = 12 − 7 → x = 5

2. Subtraction → Addition

If a term is subtracted, move it by adding.

Example: x − 4 = 9 → x = 9 + 4 → x = 13

3. Multiplication → Division

If a variable is multiplied, move the number by dividing.

Example: 5x = 20 → x = 20/5 → x = 4

4. Division → Multiplication

If a variable is divided, move the number by multiplying.

Example: x/3 = 7 → x = 7 × 3 → x = 21

Example with All Operations

Solve: (2x + 5) / 3 = 7

  1. Multiply both sides by 3: 2x + 5 = 21
  2. Subtract 5: 2x = 16
  3. Divide by 2: x = 8

Key Rules:

  • Addition → Subtract when moving
  • Subtraction → Add when moving
  • Multiplication → Divide when moving
  • Division → Multiply when moving
  • If equation simplifies to x = x → infinitely many solutions
  • If a false equality remains (e.g. 8 = 3) → no solution

Quadratic Equations

A quadratic equation has the form:

ax2+bx+c=0
  • a, b, c are constants; a ≠ 0
  • If a = 1: x₁ + x₂ = −b and x₁ · x₂ = c
  • If a ≠ 1: x₁ + x₂ = −b/a and x₁ · x₂ = c/a

Solving Methods:

  1. Factoring
  2. Quadratic Formula: x=−b±b2−4ac2a

Example 1 – Factoring: x² − 5x + 6 = 0

  1. Find two numbers that multiply to 6 and add to −5: −2 and −3
  2. Factor: (x − 2)(x − 3) = 0
  3. Solve: x = 2, x = 3

Example 2 – Quadratic Formula: 2x² − 3x − 5 = 0

  • a = 2, b = −3, c = −5
  • D = b² − 4ac = 9 + 40 = 49
  • D > 0 → 2 solutions
  • x = (3 ± 7) / 4 → x = 2.5 or x = −1

Note: if D > 0 → 2 solutions; D = 0 → 1 solution; D < 0 → no real solution

Linear Inequalities

Same as linear equations but uses: >, <, ≥, ≤ instead of =

🔹 Key rule: When multiplying or dividing by a negative number, flip the inequality sign!

Case 1 – Simple: 2x + 3 > 7

  1. Subtract 3: 2x > 4
  2. Divide by 2: x > 2

Case 2 – Negative Coefficient: −3x + 5 ≤ 2

  1. Subtract 5: −3x ≤ −3
  2. Divide by −3 (flip sign!): x ≥ 1

Case 3 – Always True: 2x + 5 > 2x + 3

Subtract 2x: 5 > 3 → always true → ✅ Solution: All real numbers (ℝ)

Case 4 – No Solution: 3x + 4 < 3x − 2

Subtract 3x: 4 < −2 → false → ❌ No solution (∅)

Quadratic Inequalities

Form: ax² + bx + c > 0 (or <, ≥, ≤)

Steps to Solve:

  1. Rearrange so one side is 0
  2. Solve the corresponding equation ax² + bx + c = 0 (find roots)
  3. Test sign of each interval
  4. Choose intervals based on the inequality symbol

Case 1 – x² − 5x + 6 > 0

  1. Roots: x = 2, x = 3
  2. Test intervals:
    • x < 2 (test x=1): positive ✓
    • 2 < x < 3 (test x=2.5): negative ✗
    • x > 3 (test x=4): positive ✓
  3. Need > 0: ✅ x < 2 or x > 3 → (−∞, 2) ∪ (3, +∞)

Case 2 – x² − 4x + 3 ≤ 0

  1. Roots: x = 1, x = 3
  2. Test intervals:
    • x < 1 (test x=0): positive
    • 1 < x < 3 (test x=2): negative ✓
    • x > 3 (test x=4): positive
  3. Need ≤ 0 (include roots): ✅ 1 ≤ x ≤ 3 → [1, 3]

Case 3 – x² + 1 < 0 → No Solution

x² + 1 ≥ 1 for all real x → always positive → can never be < 0

❌ No solution (∅)

Case 4 – x² + 1 ≥ 0 → All Real Numbers

x² + 1 is always positive → inequality always true

✅ All real numbers (ℝ)
